# Purpose

Genesis is not animated.

Genesis is choreographed.

The visual language must communicate intelligence before a single word is spoken.

Every frame should be recognisable as Gunimi.

Not because of a logo.

Because of the feeling.

---

# Visual Identity

The world of Genesis is calm.

Confident.

Intentional.

Nothing exists purely for decoration.

Every visual element communicates meaning.

---

# Design Philosophy

Minimal.

Premium.

Human.

Timeless.

Architectural.

Never futuristic for the sake of looking futuristic.

The future should feel believable.

---

# Color System

Primary

Deep graphite

Near-black backgrounds

Soft gradients

Premium whites

Secondary

Electric blue

Soft cyan

Warm violet

Rare gold accents

Never saturated neon.

Color highlights understanding.

Not technology.

---

# Light

Light represents understanding.

Darkness represents uncertainty.

Signals emit soft light.

Memory emits warm light.

AI emits almost invisible light.

Automation emits motion.

Nothing glows excessively.

---

# Materials

Glass.

Metal.

Soft shadows.

Depth.

Large breathing spaces.

No plastic.

No glossy fake reflections.

No skeuomorphism.

---

# Typography

Large.

Elegant.

Confident.

Never shout.

Never use all caps for emotional impact.

Every sentence has room to breathe.

Typography moves slowly.

---

# Motion Philosophy

Motion communicates thought.

Motion never entertains.

Animations should feel inevitable.

Nothing snaps.

Nothing bounces.

Everything glides.

---

# Motion Timing

Fast

120–180ms

Micro interactions.

Normal

250–450ms

UI transitions.

Slow

600–1200ms

Scene transitions.

Epic

2–5 seconds

Emotional reveals.

---

# Camera Language

The camera behaves like a documentary filmmaker.

Never handheld.

Never shaky.

Never spinning.

Never extreme perspective.

Camera movement should feel like curiosity.

---

# Camera Rules

The camera follows understanding.

Never follows the mouse.

Never follows UI events.

Never feels robotic.

The viewer should forget a camera exists.

---

# Depth

Layers reveal hierarchy.

Foreground

Human interaction.

Middle

Business context.

Background

AI understanding.

Depth should explain.

Never decorate.

---

# Sound Design

Silence is intentional.

Music supports emotion.

Interface sounds are subtle.

No exaggerated sci-fi sounds.

No artificial beeps.

No Hollywood technology clichés.

---

# Music

Beginning

Uneasy.

Growing tension.

Middle

Hope.

Curiosity.

Understanding.

End

Confidence.

Calm.

Resolution.

Never triumphant.

Never aggressive.

---

# Interface Motion

Panels expand.

They never pop.

Cards reveal.

They never explode.

Signals appear.

They never flash.

Memory fades in.

It never teleports.

Everything feels alive.

Nothing feels animated.

---

# Signals

Signals behave like living entities.

Soft appearance.

Gentle movement.

Natural hierarchy.

Signals never compete for attention.

Signals attract attention naturally.

---

# Memory

Memory has weight.

It appears slower than Signals.

It feels permanent.

Memory never blinks.

Memory never pulses.

Memory exists quietly.

---

# AI

AI is almost invisible.

The viewer should never think:

"There is the AI."

The viewer should think:

"The system understands."

AI is infrastructure.

Not spectacle.

---

# Scene Rhythm

Every scene breathes.

Every scene contains silence.

Every scene ends before it becomes repetitive.

Attention is earned.

Never demanded.

---

# Editing

No fast cuts.

No flashy transitions.

No cinematic clichés.

Transitions should feel like thoughts.

Not edits.

---

# Interaction

When the viewer later reaches the Living Demo...

Nothing changes.

The same animations.

The same timing.

The same typography.

The same visual language.

Genesis and the product are one experience.

---

# What Genesis Must Never Look Like

Cyberpunk.

Hollywood hacking.

Matrix.

Iron Man HUD.

Generic AI particles.

Flying code.

Random holograms.

Blue glowing brains.

Digital rain.

Robot aesthetics.

Science-fiction interfaces.

Genesis should look like software from ten years in the future.

Not science fiction.

---

# The Emotional Curve

Confusion

↓

Stress

↓

Curiosity

↓

Relief

↓

Trust

↓

Confidence

↓

Wonder

↓

Belonging

Every visual decision must reinforce this progression.

---

# Success Criteria

If every logo disappeared...

The audience should still recognise Genesis.

Because no other software feels like it.

That feeling is the Gunimi Design Language.

Genesis does not introduce a product.

Genesis introduces a new category.
