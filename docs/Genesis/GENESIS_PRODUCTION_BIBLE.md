# Voice Guidelines

Voice is calm.

Never dramatic.

Never selling.

Never shouting.

Every sentence should sound inevitable.

Voice is explaining the future.

Not advertising software.

---

# Music Guidelines

Music supports emotion.

Never dominates it.

The viewer should remember the story.

Not the soundtrack.

---

# Motion Production

Every animation is reviewed against three questions.

Does it explain?

Does it reinforce emotion?

Would removing it improve clarity?

If yes,

remove it.
